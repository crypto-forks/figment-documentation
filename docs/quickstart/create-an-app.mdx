## From the Dashboard

Click **Create New App** to create a new App and add it to the Dashboard.

<img
  src={img2}
  alt="Create New App"
  width="700"
  className="shadow--tl inline-image"
/>

Here you can:

- Give your App a name, so that you can identify it.
- Select which environment your app is in, either **Staging** or **Production**. This is a visual tag for you to label whether an app is your staging or production environment.
- Pick a protocol to use - Apps must have at least one protocol.
  - You will be able to add more protocols later, up to the limit of your App plan.

<img
  src={img1}
  alt="Create New App"
  width="400"
  className="shadow--tl inline-image"
/>

- The Dashboard will display any Apps you have created. Apps can be deleted from the **Settings** tab.

<img
  src={img3}
  alt="Dashboard Apps"
  width="400"
  className="shadow--tl inline-image"
/>

## API Key

Your API key is available on the **Overview** tab of your Dashboard when viewing an App. It is also available from the **Example** link when viewing Protocol details from the **Protocols** tab.
